A simple login and registration system built with PHP, MySQL and PHPMailer.

## Features
- User registration & login
- OTP verification via email
- Forgot / reset password
- Uses PHPMailer for emails

## How to run locally
1. Place this folder in `C:\wamp64\www\`
2. Import `user.sql` into phpMyAdmin
3. Visit: `http://localhost/Login-System`
